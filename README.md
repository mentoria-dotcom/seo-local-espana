## 📋 Schema.org para negocios locales

El Schema markup es el código que le dice a Google qué tipo de negocio es, dónde está y cuál es su teléfono. Es uno de los factores de posicionamiento local más infravalorados.

Tipos relevantes para negocios locales:
- `LocalBusiness` — base para todos los negocios locales
- `Plumber` — fontaneros
- `Electrician` — electricistas  
- `LockSmith` — cerrajeros
- `HairSalon` — peluquerías y barberías
- `Restaurant` — restaurantes y bares
- `Dentist` — clínicas dentales
- `Veterinary Care` — veterinarias

[Generador de Schema markup de Google](https://www.google.com/webmasters/markup-helper/)

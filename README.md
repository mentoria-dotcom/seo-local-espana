## 📋 Schema.org para negocios locales

El Schema markup es el código que le dice a Google qué tipo de negocio es, dónde está y cuál es su teléfono.

**Herramienta:** [Generador de Schema Markup gratuito](https://landing-five-dusky-44.vercel.app/schema-generator) — genera el JSON-LD correcto en 2 minutos.

- `LocalBusiness` — base para todos los negocios locales
- `Plumber` — fontaneros
- `Electrician` — electricistas
- `LockSmith` — cerrajeros
- `HairSalon` — peluquerías y barberías
- `BarberShop` — barberías
- `Restaurant` — restaurantes y bares
- `Dentist` — clínicas dentales
